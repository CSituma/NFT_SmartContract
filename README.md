# NFT_SmartContract

A simple Solidity smart Contract built on Etherium Network, that:
- Deploys NFT to Opensea
- Mints NFT
- Checks Balance

-https://docs.openzeppelin.com/contracts/4.x/wizard to create ERC721 Contracts for NFTS
